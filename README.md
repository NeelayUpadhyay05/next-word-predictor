# LSTM Next Word Predictor

Predicts the next word in a text sequence using LSTM neural networks. Built following a YouTube deep learning course.

## Features
- LSTM model for sequence prediction
- Trained on Kaggle text datasets (e.g., Medium articles)
- Tokenization and sequence preprocessing
- Jupyter notebook implementation

## Quick Start
1. Clone repo: `git clone [your-repo-url]`
2. Install: `pip install -r requirements.txt`
3. Download dataset from Kaggle
4. Run: `jupyter notebook lstm-next-word.ipynb`

## Tech Stack
- Python, PyTorch/TensorFlow
- Pandas, NumPy
- Jupyter, Matplotlib
